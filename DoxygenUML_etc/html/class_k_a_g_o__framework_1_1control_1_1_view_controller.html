Detailed Description

Diese Klasse kontrolliert die DrawingPanels einer ihr zugewiesenen DrawingFrame. Sie kann verschiedene Objekte erzeugen und den Panels hinzufuegen. Vorgegebene Klasse des Frameworks. Modifikation auf eigene Gefahr.
