void KAGO_framework.model.InteractiveGraphicalObject.keyPressed ( int  key)

Parameters
keyEnthält den Zahlencode für die Taste. Kann direkt aus der Klasse KeyEvent geladen werden, z.B. KeyEvent_VK_3
