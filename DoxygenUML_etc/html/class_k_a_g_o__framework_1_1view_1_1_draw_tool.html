Detailed Description

Diese Klasse dient als vereinfachte Schnittstelle zum Zeichnen. Es handelt sich um eine BlackBox fuer die Graphics2D-Klasse. Vorgegebene Klasse des Frameworks. Modifikation auf eigene Gefahr.
