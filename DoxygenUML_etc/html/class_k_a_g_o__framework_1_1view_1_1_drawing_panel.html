Detailed Description

Stellt eine Zeichenfläche in einem DrawFrame-Fenster dar. Beim DrawingPanel über die Methode "add" registrierte Objekte werden vom Panel gezeichnet. Außerdem kümmert sich das DrawingPanel um das Aufrufen der im framework realisierten Callbacks wie etwa update und draw. Diese Modellierung ist nicht sauber, da das DrawingPanel damit Funktionen eines Controllers übernimmt. Vorgegebene Klasse des Frameworks. Modifikation auf eigene Gefahr.
