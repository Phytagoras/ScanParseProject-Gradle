abstract boolean my_project.model.Scanner< TokenValue, TokenType >.scan ( String  input)
abstract

Initialisiert die tokenList neu! Scannt einen String, zerlegt ihn dabei in Tokens mit einem Typ und einem Wert gemäß den generischen Typen der Scanner-Klasse und fügt diese Tokens hinten die Tokenliste an. Im Anschluss wird der Zeiger der Tokenliste auf das erste Element gesetzt, damit ein Parser seine Arbeit beginnen kann.

Parameters
inputder zu scannende String
Returns
true, sofern der Scan komplett erfolgreich war, sonst false.
