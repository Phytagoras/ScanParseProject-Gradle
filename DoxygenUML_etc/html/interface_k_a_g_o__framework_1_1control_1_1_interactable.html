void KAGO_framework.control.Interactable.keyPressed ( int  key)

Wird einmalig aufgerufen, wenn eine Taste heruntergedrückt wird. Nach der Anschlagverzögerung löst Windows den Tastendruck dann in schneller Folge erneut aus. Eignet sich NICHT, um Bewegungen zu realisieren.

Parameters
keyEnthält den Zahlencode für die Taste. Kann direkt aus der Klasse KeyEvent geladen werden, z.B. KeyEvent_VK_3
