\doxysubsection{Detailed Description}
Diese Klasse kontrolliert die Drawing\+Panels einer ihr zugewiesenen Drawing\+Frame. Sie kann verschiedene Objekte erzeugen und den Panels hinzufuegen. Vorgegebene Klasse des Frameworks. Modifikation auf eigene Gefahr.
