Initialisiert die token\+List neu! Scannt einen String, zerlegt ihn dabei in Tokens mit einem Typ und einem Wert gemäß den generischen Typen der Scanner-\/\+Klasse und fügt diese Tokens hinten die Tokenliste an. Im Anschluss wird der Zeiger der Tokenliste auf das erste Element gesetzt, damit ein \mbox{\hyperlink{interfacemy__project_1_1model_1_1_parser}{Parser}} seine Arbeit beginnen kann.
